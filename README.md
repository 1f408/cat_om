# cat\_om(CAT's atOM feed)

[cats\_dogs](https://github.com/1f408/cats_dogs) のドキュメントを git で管理している場合において、  
ドキュメントの更新情報から Atom Feed[^1] を生成する。

## 使い方

1. Install
  - `sh build.sh`
2. Help
  - `./cat_om -h`

使い方は、Help の Usage を参照。
- [設定ファイルについて](./src/catom/README.md)
